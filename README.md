# vcodes-installer
This Installer installing Vcodes V4 to Ubuntu VPS

> # **How To Install?**

```js
//Step One
apt install curl
//Step Two
curl -o installer.sh https://raw.githubusercontent.com/kardespro/vcodes-installer/main/installer.sh
//Step Three
chmod +x installer.sh
//Step Four
$ ./installer.sh
```

> # **RUN It ! **

```bash
cd /Vcodes/benedict/
chmod +x start.sh
$ ./start.sh
```
